**Branch**: `004-design-system-legacy-pages` | **Date**: 2026-07-19 | **Spec**: [spec.md](./spec.md)

**Input**: Feature specification from `/specs/004-design-system-legacy-pages/spec.md`

## Summary

Complete the client design system rollout started in `003-client-design-system` by migrating the **remaining legacy routes** (Artist dashboard, Artist profile, Checkout, Admin users) and their shared components to `DesignSystemLayout` and existing UI primitives. Remove `LegacyLayout`, set global `index.css` body to canvas/ink, and drop unused legacy Tailwind color tokens. **Client-only** — no API, schema, or routing changes.
